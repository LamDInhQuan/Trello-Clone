# Trello-Clone – Website Quản lý Công Việc (Full-Stack)

## Công nghệ
Java, Spring Boot, Spring Security, JWT, MongoDB, WebSocket, RESTful API, ReactJS, Redux

## Mô tả dự án
Ứng dụng quản lý công việc mô phỏng Trello, cho phép người dùng tạo **board – column – card**, kéo thả card giữa column, phân quyền người dùng theo vai trò (**owner, member, guest**), gửi **thông báo thời gian thực** và xác thực người dùng bằng **JWT**.

## Vai trò & Nhiệm vụ

### Frontend
- Phát triển UI kéo thả với ReactJS & Redux  
- Quản lý state toàn cục, gọi API, bảo vệ route theo vai trò  
- Tối ưu trải nghiệm người dùng (UX) và component-based design  

### Backend
- Thiết kế RESTful API, xác thực & phân quyền bằng Spring Security + JWT  
- Lưu trữ dữ liệu MongoDB, xử lý thời gian thực bằng WebSocket  
- Triển khai kiến trúc backend chuẩn, xử lý logic nghiệp vụ và tối ưu truy vấn  

## Điểm nổi bật
- UI drag & drop mượt mà, trải nghiệm tương tự Trello  
- Phân quyền chi tiết: owner, member, guest  
- Cập nhật real-time cho mọi thay đổi trong board  
- Triển khai full-stack production-ready (frontend + backend)
